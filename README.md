# Champion Shipping Logistics

A lightweight Next.js + Tailwind CSS starter for a logistics website built for Vercel deployment.

## What is included

- `src/pages/index.jsx` — Home page
- `src/pages/services.jsx` — Services overview
- `src/pages/pricing.jsx` — Pricing page
- `src/pages/contact.jsx` — Contact page
- `src/components` — reusable UI components
- `src/styles/globals.css` — global styling and brand variables
- `public/logo.svg` — logo placeholder

## Brand palette

- Primary: `#1E3A8A`
- Secondary: `#2563EB`
- Accent: `#F59E0B`
- Text / Dark: `#1F2937`
- Secondary text: `#374151`
- Background: `#FFFFFF` / `#F7F7F7`

## Setup

```bash
npm install
npm run dev
```

## Deploy

This project is ready for Vercel. Use `npm run build` to verify production build.
