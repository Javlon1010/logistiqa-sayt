# Brand Kit & Project Brief

## Brand palette

- Indigo / cyber-blue: `#1E3A8A`
- Costa blue: `#2563EB`
- Smooth silver / dark: `#1F2937`
- Accent (youth): `#F59E0B`
- Background: `#FFFFFF` or `#F7F7F7`
- Secondary text: `#374151`

## Typography

- Heading: Inter, 700
- Body: Inter, 400-500

## Logo usage

- SVG format
- Header left logo with CTA visible on larger screens
- Responsive placement for center / right CTA buttons on mobile

## Sitemap

- Home

  - Global Logistics Solutions

## Content blocks

Home:
- Hero with CTA buttons “Konsultatsiya oling”, “Xizmatlar”
- About intro
- Service cards
- Testimonials carousel
- Trust badges
- Short contact form
- Footer with contact, social, sitemap

Services:
- Service cards with title, description, benefits, icon, CTA
- Pricing/consultation prompt

Pricing:
- Tariff grid: Standart, Global, Maksimal
- Request form if pricing is dynamic
- FAQ section

Contact:
- Contact form
- Google Maps / location
- Social links and contact details

## SEO structure

- `/services/freight-shipping`
- `/pricing`
- `/about-us`
- `/contact`

Meta descriptions and Open Graph values should be concise and use keywords:
shipping, logistics, freight, warehousing, customs, global logistics.

## Development stack

- Frontend: Next.js + Tailwind CSS
- API: Next.js serverless routes
- CMS: Optional headless CMS integration later
- SEO: SSR pages + meta templates
- Analytics: Google Analytics 4 + Yandex Metrika
- Performance: image optimization, lazy loading, code splitting
